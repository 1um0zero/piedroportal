# Attachments

Drop here any external material that feeds a tracker task: client emails (.eml/.md),
spec PDFs, screenshots, exported CSVs, DNS records, DPA copies, etc.

**Convention**
- One sub-folder per tracker topic number, e.g. `09-compliance/`, `06-email/`.
- Reference a file from the tracker with a relative link, e.g.
  `📎 [DPA-Anthropic.pdf](attachments/09-compliance/DPA-Anthropic.pdf)`.
- For emails, save as `YYYY-MM-DD-subject.eml` (or paste the text into a `.md`).

The tracker (`docs/PROJECT-TRACKER.md`) is the index; this folder is the evidence store.
